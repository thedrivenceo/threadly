## What is Post Companion?

Post Companion is a content scheduling and publishing platform for Meta's Threads. It combines AI-powered content repurposing, smart scheduling, engagement analytics, brand voice customization, and a Chrome extension into a single tool for content creators.

## Core capabilities

- **Content Bank**: Store drafts, categorize by pillar and content type, schedule or post directly
- **AI Repurposing**: Turn articles, voice notes, and documents into short, mid, and long form posts using your brand voice
- **Smart Scheduling**: Set your posting frequency, and Post Companion finds the best times from your engagement data
- **Multi-Account**: Connect multiple Threads accounts (up to 10 on Team plan)
- **Brand Voice**: Upload a brand doc or answer style guide questions so AI writes like you
- **Chrome Extension**: Save posts from Threads and any website to your Inspiration board
- **Analytics**: Real Threads data with views, engagement, best time heatmap, and recommendations
- **Team Management**: Invite team members with Owner, Manager, and Member roles

## Tech stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + TypeScript + Vite + Tailwind CSS |
| UI | shadcn/ui + Framer Motion |
| Backend | Supabase (Auth, Database, Edge Functions) |
| AI | Claude API (Haiku model) |
| Social API | Threads API (OAuth, publishing, insights) |
| Hosting | Vercel |
| Font | Figtree |
