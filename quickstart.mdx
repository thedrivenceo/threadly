## Prerequisites

Before you start, make sure you have:

- Node.js 18+ and npm
- A Supabase account and project
- A Meta Developer account with a Threads app configured
- An Anthropic API key for Claude

## Setup

<Steps>
  <Step title="Clone the repository" icon="git-branch" title-type="p">
    ```bash
    git clone <your-repo-url>
    cd threadflow
    npm install
    ```
  </Step>

  <Step title="Configure environment variables" icon="settings" title-type="p">
    Create a `.env` file in the project root. Never commit this file.

    ```bash title=".env"
    VITE_SUPABASE_URL=https://xaoodbhmrkpcweedvoev.supabase.co
    VITE_SUPABASE_ANON_KEY=your_anon_key
    ```

    <Callout kind="danger">
      Never hardcode API keys in source files. All secrets must stay in `.env` or Supabase secrets.
    </Callout>
  </Step>

  <Step title="Set Supabase secrets" icon="lock" title-type="p">
    These secrets are required by Edge Functions. Set them via the Supabase CLI or dashboard.

    ```bash
    supabase secrets set ANTHROPIC_API_KEY=your_anthropic_key
    supabase secrets set THREADS_APP_ID=your_meta_app_id
    supabase secrets set THREADS_APP_SECRET=your_meta_app_secret
    supabase secrets set THREADS_REDIRECT_URI=https://xaoodbhmrkpcweedvoev.supabase.co/functions/v1/threads-auth-callback
    supabase secrets set WEBHOOK_SECRET=your_webhook_secret
    ```
  </Step>

  <Step title="Start the dev server" icon="play" title-type="p">
    ```bash
    npm run dev
    ```

    The app will be available at `http://localhost:5173`.
  </Step>

  <Step title="Connect your Threads account" icon="at-sign" title-type="p">
    1. Sign up or log in to Threadflow
    2. Go to **Settings** and click **Connect Threads**
    3. Authorize the app via Meta's OAuth flow
    4. Your Threads profile data will sync automatically
  </Step>
</Steps>

## Verify the setup

After connecting your Threads account, confirm everything works:

| Check | How to verify |
|-------|--------------|
| Auth | You can sign up, log in, and see the dashboard |
| Threads connection | Your Threads username appears in Settings |
| AI generation | Create a post and use the "Generate with AI" option |
| Scheduling | Schedule a post and confirm it appears in the calendar |
| Publishing | Publish a test post directly to Threads |
